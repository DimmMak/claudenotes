# 💀 PITFALLS — Mistakes I Made, Lessons I Keep

> Personal post-mortems. When something goes wrong using ClaudeNotes (or while studying via the system), log it here. Honest, dated, brief.
>
> **Rule:** every entry has 3 parts — what happened, why it sucked, what I'll do differently.
>
> When 3 recurring lessons appear → promote them into a `PLAYBOOK.md` for situational strategies.

---

## 🎯 Why this file exists

Universal "never do" rules live in `PREFERENCES.md`. **PITFALLS is for YOUR specific mistakes** — the ones nobody could've predicted, the ones you only learn the hard way.

This is the file that makes future-Danny smarter than past-Danny.

---

## 📋 Entry template

```markdown
## YYYY-MM-DD — Short title

**What happened:** Concrete description in 1-2 sentences.
**Why it sucked:** The actual cost. (Lost time? Lost notes? Wrong file?)
**What I'll do differently:** One specific behavior change. Not a vague "be more careful."
**Tags:** #related-to-what (e.g., #capture, #sort, #lecture, #burnout)
```

## 📈 When to promote a pattern

If you log the same lesson 3+ times under different dates → **that's a recurring failure mode.** Promote it:

1. Create `PLAYBOOK.md` if it doesn't exist
2. Write a "When [situation] happens, do [strategy]" entry there
3. In PITFALLS, link to the playbook entry: `→ See PLAYBOOK.md "When X"`

That's how PLAYBOOK gets EARNED, not pre-created.

---

## 🚫 What does NOT belong here

- ❌ Universal rules ("never delete files") → `PREFERENCES.md` "NEVER DO"
- ❌ Daily activity ("studied module 2 today") → `git log` or `/capture`
- ❌ Tactical sequences ("capture → sort → recall") → `COMBOS.md`
- ❌ Reference material ("the 19 patterns are...") → `MEMORIZE.md`

PITFALLS is **only** for: *"I did X, it backfired, here's what I learned."*

---

## 🥊 The honest truth

> If this file stays empty for 30 days, you're either perfect or not paying attention. Probably the latter.

Set yourself a Sunday review: when you run `/notes evolve`, also glance here. **Empty PITFALLS this week? What did you actually learn?**
